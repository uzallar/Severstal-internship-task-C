# Severstal-internship-task-C
## Класс DataPreprocessor на Python
Python-класс для автоматической предобработки табличных данных в формате pandas DataFrame. 
Решение находится в файле **`data_preprocessor.py`**
Класс выполняет три основные операции:

### 1. Обработка пропущенных значений
- Удаляет столбцы с долей пропусков > порогового значения
- Заполняет оставшиеся пропуски (медианой/средним/модой)

### 2. Кодирование категориальных признаков  
- One-hot кодирование для строковых столбцов
- Ограничение по количеству уникальных категорий

### 3. Нормализация числовых признаков
- Min-Max нормализация (по умолчанию)
- Стандартизация (z-score)

## Параметры обработки

### Пропуски (missing)
- **threshold** (0-1): порог для удаления столбцов (по умолчанию 0.5)
- **strategy**: 'mean', 'median' или 'mode' (по умолчанию 'median')
### Категории (encode)
- **max_categories**: максимальное количество категорий для кодирования (по умолчанию 10)
- **drop_first**: удалять первую категорию (по умолчанию True)
### Нормализация (normalize)
- **method**: 'minmax' или 'std' (по умолчанию 'minmax')
- **exclude_cols**: список столбцов для исключения
